## ✨ What Is This?

GA Real Estate Academy is a full-stack web application that helps aspiring Georgia real estate agents pass the PSI licensing exam. It combines a structured 26-level curriculum, AI-generated practice questions, live exam simulations, and a gamified progress system — all powered by a free-tier AI backend.

## 🛠️ Tech Stack

| Layer | Technology |

## 🏁 Quick Start

### Prerequisites
- Node.js 18+
- PostgreSQL database (free: [neon.tech](https://neon.tech))
- OpenRouter API key (free: [openrouter.ai](https://openrouter.ai))

### Setup

```bash
# 1. Clone the repo
git clone https://github.com/pitchiluxe/GARealState.git
cd GARealState

# 2. Install dependencies
npm install

# 3. Create environment file
cp .env.example .env.local
# Edit .env.local with your keys

# 4. Push schema & seed demo data
npm run db:push
npm run db:seed

# 5. Start the dev server
npm run dev
# → http://localhost:4000
```

## ⚙️ Environment Variables

```env
# OpenRouter AI (free tier available)
ANTHROPIC_AUTH_TOKEN=sk-or-...
ANTHROPIC_BASE_URL=https://openrouter.ai/api
ANTHROPIC_MODEL=deepseek/deepseek-v4-flash:free

# Database (PostgreSQL)
DATABASE_URL=postgresql://user:pass@host/dbname?sslmode=require

# Auth
NEXTAUTH_SECRET=your-random-secret
NEXTAUTH_URL=https://your-domain.vercel.app

# App
NEXT_PUBLIC_APP_NAME="GA Real Estate Academy"

# LiveKit — optional, for Study Group voice/video
LIVEKIT_API_KEY=
LIVEKIT_API_SECRET=
LIVEKIT_URL=
NEXT_PUBLIC_LIVEKIT_URL=
```
